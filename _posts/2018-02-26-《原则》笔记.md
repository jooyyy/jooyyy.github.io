```
不管我一生中获得了多大的成功，其主要原因，都不是我知道多少事情，而是我知道在无知的情况下自己应该怎么做。
```

```
假如没有原则，我们将被迫针对省中遇到的各种难以预料之事孤立地做出反应，就好像我们是头一次碰到这些事一样。
```

```
独立思考并决定：（1）你想要什么；（2）事实是什么；（3）面对事实，你是如何实现自己的愿望
```
